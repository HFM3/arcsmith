<h2 class="as-hero__title">Cite ArcSmith</h2>
<p class="as-hero__sub">If ArcSmith supports your research or published work, a citation is appreciated. It is not required under the MIT license, but documented use builds trust in the tool.</p>
</div>

Pick a format below. Each is generated for the current release. Run
`pip show arcsmith` to confirm the version you used.

## APA

```text
Mros, H. F., III. (__ARCSMITH_YEAR__). ArcSmith: ArcPy toolbox utilities [Computer software]. Version __ARCSMITH_VERSION__. https://pypi.org/project/arcsmith/
```

## Chicago

```text
Mros, Henry F., III. __ARCSMITH_YEAR__. ArcSmith: ArcPy Toolbox Utilities. Version __ARCSMITH_VERSION__. Computer software. https://pypi.org/project/arcsmith/
```

## BibTeX

```bibtex
@software{ArcSmith,
  author  = {Mros, III, Henry F.},
  title   = {ArcSmith: ArcPy Toolbox Utilities},
  year    = {__ARCSMITH_YEAR__},
  version = {__ARCSMITH_VERSION__},
  url     = {https://pypi.org/project/arcsmith/}
}
```
